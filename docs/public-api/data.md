# Data widgets

Data widgets provide Material-style helpers for item views, tables, trees, lists, and structural separators.

## Public headers

```cpp
#include <qtmaterial/widgets/data/qtmaterialdivider.h>
```

## Divider

QtMaterialDivider provides a lightweight Material separator for layouts, panels, lists, and forms.

```cpp
auto* divider = new QtMaterial::QtMaterialDivider(parent);
```

Use dividers to separate related content groups without introducing a full surface or card.
