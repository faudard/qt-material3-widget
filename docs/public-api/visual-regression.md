# Visual regression testing

Slice 9 adds a visual regression harness for the theming subsystem and Theme Studio.

## Goals

The harness protects the resolved visual contract across:

- light and dark mode
- standard and high contrast
- seed-color changes
- focus-ring and accessibility-token changes
- component preview rendering
- Theme Studio screenshot export

## Test levels

### Token-board goldens

Token-board images are generated by a deterministic painter instead of platform-native widgets. They are the safest default regression gate because they avoid most cross-platform font and rasterization noise.

### Component-grid artifacts

Component-grid rendering uses real QtMaterial widgets. These tests always produce artifacts and verify basic rendering health. Pixel comparison is opt-in because different platforms may produce small pixel differences.

## Updating goldens

```bash
scripts/update-visual-goldens.sh build
```

or manually:

```bash
QTMATERIAL3_UPDATE_VISUAL_GOLDENS=1 \
ctest --test-dir build --output-on-failure -R tst_theme_visual_regression
```

## Strict comparison

```bash
QTMATERIAL3_VISUAL_STRICT=1 \
ctest --test-dir build --output-on-failure -R tst_theme_visual_regression
```

Optional small tolerance:

```bash
QTMATERIAL3_VISUAL_STRICT=1 QTMATERIAL3_VISUAL_MAX_DIFF=25 \
ctest --test-dir build --output-on-failure -R tst_theme_visual_regression
```

## Artifacts

By default, artifacts are written next to the test executable in `visual-artifacts`.

Override this with:

```bash
QTMATERIAL3_VISUAL_ARTIFACTS_DIR=/tmp/qtm3-visual \
ctest --test-dir build --output-on-failure -R tst_theme_visual_regression
```

The artifact directory contains:

- `*.actual.png`
- `*.diff.png` for failures
- `manifest.json` with dimensions and SHA-256 hashes

## Merge policy

Golden changes should be reviewed when one of these changes intentionally occurs:

1. color generation changed
2. token defaults changed
3. component specs changed
4. shape/elevation/typography defaults changed
5. accessibility/focus-ring behavior changed

Do not update goldens to hide accidental rendering drift.
